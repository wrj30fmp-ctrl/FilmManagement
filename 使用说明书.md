# 胶片管理器 — 使用说明书

## 快速开始

1. 双击桌面上的「胶片管理器」图标，或运行 `dist/FilmManager/FilmManager.exe`
2. 程序会自动在 `文档/FilmManager/` 下创建数据库
3. 左侧导航栏点击各功能页面，点击左上角 ☰ 按钮可以折叠/展开侧栏

---

## 各功能说明

### 📦 胶卷库存

管理你拥有的所有胶卷。

**新增库存**：点击绿色 `+ 新增库存` 按钮 → 填写信息 → 确定
- 型号、画幅为必填项（标红色 * 号）
- 有效期、价格可后续补充
- 可以上传胶卷盒照片（点击虚线框选择图片）

**筛选**：按画幅、过期状态筛选，或直接输入关键词搜索

**操作**：选中一行后可以编辑、删除、生成标签

---

### 📷 拍摄记录

记录每一卷胶卷的拍摄过程。

**从库存开始拍摄**（推荐）：
1. 点击橙色 `从库存开始拍摄` 按钮
2. 选择要使用的库存胶卷
3. 填写相机、镜头等信息 → 确定
4. 库存数量自动减 1，新建一条拍摄记录

**手动新增**：点击绿色 `+ 新增记录`，手动填写信息（不消耗库存）

**管理状态**：
- 选中一条记录 → 点 `▶ 推进状态` → 状态前进一格
- 例如：LOADED → SHOOTING → SHOT（已拍摄，待冲洗）
- 可点 `◀ 退回状态` 撤销

**批量操作**：Ctrl+点击多选 → 点 `批量推进状态` 一键推进

**读取 EXIF**：编辑拍摄记录时 → 点紫色 `📷 从照片读取 EXIF` → 选择照片 → 自动填入相机、镜头、ISO

---

### 🧪 冲洗管理

记录胶卷的冲洗过程。**使用前提**：拍摄记录的状态必须先推进到「已拍摄，待冲洗」。

1. 在「拍摄记录」中将胶卷状态推进到「已拍摄，待冲洗」
2. 打开「冲洗管理」页面 → 选中记录 → 点击 `填写 / 编辑冲洗信息`
3. 填写冲洗方式、工艺、费用、日期等

---

### 🖼 扫描管理

记录扫描信息。**使用前提**：拍摄记录的状态必须先推进到「已冲洗」。

1. 打开「扫描管理」页面 → 选中记录 → 点击 `填写 / 编辑扫描信息`
2. 填写扫描设备、分辨率、文件格式
3. 点击 `选择...` 指定扫描文件夹路径
4. 保存后可以随时点击 `📂 打开扫描文件夹` 快速访问

---

### 📁 归档管理

记录底片存放位置。**使用前提**：拍摄记录的状态必须先推进到「已扫描」。

1. 打开「归档管理」页面 → 选中记录 → 点击 `填写 / 编辑归档信息`
2. 填写底片存放位置、底片册编号、是否已云端/异地备份

---

### 📊 数据统计

查看费用分析、拍摄趋势、相机使用频率等图表。页面打开时自动刷新。

- **费用构成**：饼图显示胶卷购买、冲洗、扫描的费用占比
- **月度消费趋势**：折线图显示每月大概花了多少钱
- **库存消耗预测**：根据历史拍摄速度，预测库存还能用多久

---

### ⚙ 设置

**备份**：点击 `立即备份` 手动备份，或开启启动/退出时自动备份。备份文件保存在 `文档/FilmManager/backups/`

**恢复**：点击 `恢复备份...` 选择之前的备份文件，恢复前会自动备份当前数据

**导出**：
- CSV：每个表单独导出为 CSV 文件（UTF-8 编码，Excel 可直接打开）
- Excel：所有数据导出到一个 Excel 文件，每个表一个 Sheet

**导入**：
- 点击 `从 CSV 导入库存...` → 选择 CSV 文件 → 预览数据 → 确认导入
- 可先 `下载 CSV 模板` 查看正确格式
- 列名支持中英文（如"品牌"和"brand"均可）
- 必填字段：型号、画幅

**标签打印**：
- 点击 `批量生成库存标签` → 生成 HTML 文件 → 浏览器打开 → 直接打印
- 在库存页面也可以单选生成标签

---

## 数据安全

- 所有数据存储在 `C:\Users\<用户名>\Documents\FilmManager\film_manager.db`
- 程序启动和退出时自动备份（可在设置中开关）
- 删除操作为"软删除"，未来可恢复
- CSV/Excel 导出可用于数据迁移

## 常见问题

**Q: 冲洗/扫描/归档页面没有记录？**
A: 这些页面只显示特定状态的胶卷。需要先在「拍摄记录」中将状态推进到对应阶段。工作流：库存 → 拍摄 → 冲洗 → 扫描 → 归档。

**Q: 从哪里开始？**
A: 推荐流程：① 库存页添加胶卷 → ② 拍摄页「从库存开始拍摄」→ ③ 推进状态 → ④ 冲洗/扫描/归档页填写信息。

**Q: 如何迁移到新电脑？**
A: 复制 `文档/FilmManager/` 整个文件夹到新电脑同位置，或使用备份恢复功能。
